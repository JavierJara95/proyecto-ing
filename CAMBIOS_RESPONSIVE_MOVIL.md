# Cambios responsive para móvil/smartphone

Se corrigieron las páginas HTML para que sean visibles correctamente en pantallas pequeñas, manteniendo el formato visual ya establecido.

## Ajustes aplicados

- Corrección del menú superior para que no se corte en celulares.
- Logo adaptable al ancho del teléfono.
- Formularios en una sola columna en smartphone.
- Botones al 100% del ancho disponible en móvil.
- Tarjetas y módulos del dashboard adaptados a una columna.
- Tablas con scroll horizontal controlado para evitar que rompan la pantalla.
- Modales y popups ajustados al alto/ancho del celular.
- Corrección de desbordes horizontales en resultados, folios y bloques de texto.

## Archivos modificados

- `smaa-frontend/css/styles.css`

No se cambió la estructura base del proyecto ni la lógica de roles previamente implementada.
